# GROQI
A modern AI chatbot built using React and Node.js, powered by LLaMA-3 via the Groq API. Features a responsive UI, real-time messaging, and well-structured AI responses.
This project is a full-stack AI chatbot application developed using React (frontend) and Node.js with Express (backend).

The chatbot leverages the Groq API to access LLaMA-3.1 models, enabling fast and intelligent responses. The application supports real-time user interaction, markdown-formatted AI responses, smooth UI animations, and a clean chat interface.

The backend securely handles API requests and environment variables, while the frontend provides a modern, responsive user experience. This project demonstrates practical usage of AI APIs, RESTful communication, and full-stack development concepts.

### Tech Stack
- Frontend: React, Tailwind CSS
- Backend: Node.js, Express
- AI Model: LLaMA-3.1 (via Groq API)
- API Communication: REST (fetch)
- Environment Management: dotenv

- ### Features
- AI-powered conversational chatbot
- Fast responses using Groq LLMs
- Clean and responsive UI
- Markdown-formatted AI answers
- Typing indicator for better UX
- Secure API handling using environment variables
